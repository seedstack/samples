# W20 hello world sample

An "Hello World!" W20 project demonstrating how to initialize the framework and display a trivial HTML view.

# Running it

    npm install
    bower install
    grunt connect

# Usage

Just point your favorite browser to the following URL:

    http://localhost:9001
